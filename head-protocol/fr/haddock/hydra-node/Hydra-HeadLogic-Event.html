data Event tx Source #

The different events which are processed by the head logic (the "core"). Corresponding to each of the "shell" layers, we distinguish between events from the client, the network and the chain.

Event received from peers via a Hydra.Network.

  • ttl is a simple counter that's decreased every time the event is reenqueued due to a wait. It's default value is defaultTTL
